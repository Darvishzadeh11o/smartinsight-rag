PDF-RAG Assistant

A Retrieval-Augmented Generation (RAG) application that allows users to upload any PDF and ask intelligent questions about its content.
The system extracts text, creates vector embeddings using OpenAI, stores them in a FAISS index, and returns accurate, source-based answers through a clean Streamlit interface.
